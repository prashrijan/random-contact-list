## Introduction

The **Random Contact List App** simulates a phone interface with a contacts list fetched dynamically from an external API. It includes interactive UI features like unlocking the screen, searching for contacts, and viewing detailed contact information.

---

## Live Link

https://prashrijan.github.io/random-contact-list/

## Features

- Simulates a mobile phone interface with lock, app, and contact screens.
- Fetches random user data from [Random User API](https://randomuser.me).
- Search functionality for filtering contacts.
- Accordion-style UI for displaying detailed contact information.
- Responsive design.

---

## Technologies Used

- **Frontend Framework**: Bootstrap 5
- **Icons**: Bootstrap Icons
- **Languages**: HTML, CSS, JavaScript
- **API**: Random User API
